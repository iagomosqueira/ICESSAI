Steps to MSE
==========

- Define and agree on **objectives** & **limits**

- Translate them into **performance indicators**

- Identify appropriate **Management Procedures**

- Define a set of **Operating Models**

- Conduct **simulations**

- **Summarize** performance

- Select **best MP**
